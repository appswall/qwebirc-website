Downloading using git

You can check out the source code using git: 
git clone https://github.com/qwebirc/qwebirc.git <directory>
You can then update to new versions just by typing: 
git pull

File downloads

We strongly recommend you check the source code out using git, this will allow you to update to later versions trivially, if you insist on downloading the following files are provided.
