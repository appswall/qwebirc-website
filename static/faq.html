Installation

What do I need to run qwebirc?

You need Python (at least version 2.5) and Twisted (at least version 8.2.0).
Though qwebirc will work without them, it is highly recommended that you also have Java and Mercurial.

Where do I begin?

  • Copy config.py.example to config.py, and edit it to fit your setup.
  • Run compile.py to minify the Javascript/CSS and copy it to the correct location.
  • Run run.py (if you get an error about the select reactor being already registered just run it again).

I get warnings while running compile.py about java or hg!

You can ignore those, but if you're seriously looking to deploy qwebirc on a public site you really should install Java to minimise the size of your JavaScript, and Mercurial for tagging purposes (to minimise traffic and optimise for caches).
